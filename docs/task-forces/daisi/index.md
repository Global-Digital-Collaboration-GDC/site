# DAISi

## Under development

The DAISi Task Force materials are currently being prepared and will be published here as they are ready.

## Scope

The DAISi Task Force focuses on device APIs and common hardware security requirements for digital identity wallet solutions, with an emphasis on interoperability, high-assurance deployments, and reduced dependency on proprietary device ecosystems.
