# Participation Guide

## Who this is for

This guide is for organizers, advisors, contributors, and invited participants who help shape the event program and supporting outputs.

## Expected contributions

Participants are encouraged to:

- attend relevant meetings or review notes asynchronously
- keep action items current
- propose edits directly in GitHub where possible
- raise blockers early and document needed decisions

## Ways to contribute

### Update the site

Edit Markdown files in this repository when a decision or durable update should be published.

### Use the wiki

Use the [site repository wiki](https://github.com/Global-Digital-Collaboration-GDC/site/wiki) for evolving notes, draft agendas, and workshop material.

### Open issues or pull requests

GitHub issues can track work, and pull requests provide reviewable changes to formal content.
