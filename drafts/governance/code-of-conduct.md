# Code of Conduct

## Status

Draft placeholder.

Global Digital Collaboration 2026 aims to create a respectful, inclusive, and constructive environment for all participants.

## Expectations

Participants should:

- engage respectfully across sectors and geographies
- assume good intent while addressing impact directly
- avoid harassment, discrimination, and personal attacks
- respect confidentiality boundaries when a meeting or document requires them

## Reporting

Until a formal reporting path is published, concerns should be routed to the designated organizer leads and documented in an appropriate, privacy-conscious way.
